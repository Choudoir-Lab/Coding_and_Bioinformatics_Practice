
<br>

*Built-in string functions*

`.split()`

Creates a list of items by splitting a string at a specified character.

``` python
string1 = "1.2.3.4.5.6"

string2 = string1.split(".")
```

Output:

``` python
print(string2)

["1", "2", "3", "4", "5", "6"]
```
